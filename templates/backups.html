Backups and restore

The app creates a backup before changing a MAC list. You can also create a manual backup from a WiFi network page. Download saves a backup file. Restore replaces the current MAC list with the MAC list saved in the selected backup.

Restore warning

Restoring a backup replaces the current MAC list with the MAC list saved in that backup. Use this only to undo a bad change.
